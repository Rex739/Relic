# CoralOS Local Configuration

This directory is reserved for non-secret local configuration notes for the Relic CoralOS governance proof.

Do not commit:

- Coral Server auth keys
- API keys
- wallet values
- seed phrases
- private keys
- Fetch/uAgents seeds
- Vercel or Render secrets

The first demo uses only non-secret options passed through Coral Server:

- `RELIC_API_BASE_URL`
- `RELIC_REVIEW_REQUEST`
- `RELIC_WORKSPACE_URL`
